# Changelog

## v1.0.0

- Updated the real WinForms app to match the GitHub Pages mockup more closely.
- Added the app logo to the native title bar and Windows taskbar.
- Replaced native tab chrome with flat in-app navigation.
- Refined buttons, work surfaces, grids, and comparison status coloring.
- Fixed the AD group and folder rights domain/controller field wiring.

## v0.1.0

- Initial AD Access Reporter release.
- Added multi-group AD user lookup with nested membership support.
- Added group comparison grid and CSV export.
- Added folder/file NTFS ACL reporting for local and UNC paths.
- Added optional AD group expansion for folder ACL entries.
- Added GitHub Pages site, release manifest, portable zip, and checksums.
