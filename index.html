AD Access Reporter

A Windows desktop utility for pulling Active Directory group users, comparing multiple groups, exporting CSVs, and reporting folder NTFS rights from local or network share paths.

Download EXE Portable ZIP GitHub
AD group exports

Enter one or more group names, include nested members when needed, and export the full user list to CSV.

Group comparison

Compare several groups at once and see who is common, unique, or missing from each group.

Folder rights

Scan NTFS ACLs on local, file, or UNC paths and optionally expand AD groups into user rows.
